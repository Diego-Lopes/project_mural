# Projeto Mural de notícias
---
*Um projeto em nodejs com renderização de página estática.*

```bash
  *Para iniciar o projeto
  npm install 
```

Este projeto é feito com base do curso ***[ProgramadorBR](https://programadorbr.com/)***

- [x] POST
